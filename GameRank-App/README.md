# GameRank: Plataforma de Exploración de Videojuegos

Este proyecto es una aplicación web interactiva desarrollada para la asignatura de **Servicios y Aplicaciones de Redes de Ordenadores**. Permite a los usuarios explorar, puntuar y seguir sus videojuegos favoritos.

## Tecnologías utilizadas
- **Lenguaje:** Python
- **Fuentes de datos:** Procesamiento de archivos XML y consumo de APIs REST externas (FreeToGame/MMOBomb).
- **Funcionalidades:** - Autenticación de usuarios (login/registro).
  - Sistema de puntuación (0-5) y comentarios con marca de tiempo.
  - Listas personalizadas de juegos seguidos y votados.

## Instalación y Ejecución
1. Asegúrate de tener Python instalado.
2. Instala las dependencias necesarias: `pip install -r requirements.txt`.
3. Ejecuta la aplicación: `python main.py` .
4. Accede a `http://localhost:5000` en tu navegador.
